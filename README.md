# Kircraft Bot Prototype

Интерактивный прототип интерфейса Telegram-бота мастерской **Kircraft**.

Репозиторий содержит пользовательский интерфейс и навигацию бота, реализованные в виде веб-прототипа. Проект используется для проверки пользовательских сценариев, визуальной проработки интерфейса и демонстрации поведения системы до реализации полноценного Telegram-бота.

---

## Назначение репозитория

Прототип предназначен для:

- визуальной проработки интерфейса Telegram-бота  
- проверки пользовательских сценариев  
- демонстрации пользовательского потока  
- согласования поведения интерфейса перед разработкой  

Репозиторий **не является production-реализацией бота** и не содержит серверной логики.

Системная документация и описание сценариев системы находятся в отдельном репозитории:

**kircraft-system**

---

## Реализованные пользовательские сценарии

В прототипе реализованы основные пользовательские потоки бота:

- просмотр каталога изделий  
- просмотр изделий по коллекциям  
- просмотр новинок  
- просмотр карточки изделия  
- выбор параметров изделия  
- добавление товара в корзину  
- управление корзиной  
- оформление заказа  
- отображение состояний заказа  

Прототип моделирует пользовательский опыт взаимодействия с Telegram-ботом.

## Запуск проекта

Установка зависимостей:
pnpm install

Запуск dev-сервера:
pnpm dev

После запуска приложение будет доступно по адресу:
http://localhost:3000
